# A Rock Paper Scissors Game

## Some context

  This repo is an exercise from Odin Project about a brief and training for basics of Javascript by making a Rock Paper Scissors Game.

## Skills Demonstrated

- **Learning Javascript functions and systems:** Successfully demonstrated a how to define functions, pass arguments, and the connection between them.

- **Control Flow & Conditional Logic:** Utilized conditional logic to decide the winner of each round

- **Randomization & Automation:** Implemented Randomized logic to make the computer decide the moves for each round

- **State Management & Variable Scope:** Constructed a State Management and Variables to save round numbers and score numbers

- **Input Validation & String Manipulation:** Applied a String Manipulation to make the human input clean and understandable by the codes.

- **Version Control:** Practiced disciplined Git workflow with regular, descriptive commits documenting style changes and structural adjustments.
